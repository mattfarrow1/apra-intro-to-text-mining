➜ N-Grams

What are n-grams?

  • A continuous sequence of n words
  • Can be used ‘as is’, or run through a stemmer to get morphemes
  • Offers more context than single words

Set up bigrams

# Unnest into bigrams
tidy_bigrams <- reviews |> 
  unnest_tokens(bigram, 
                text, 
                token = "ngrams", 
                n = 2) |> 
  filter(!is.na(bigram))
